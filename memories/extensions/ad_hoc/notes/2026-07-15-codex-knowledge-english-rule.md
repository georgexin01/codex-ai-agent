# Codex knowledge language rule

For this workspace, write new durable `.codex` knowledge, skill updates, Pinia contract audits, migration summaries, and generated developer guidance in English.

Do not translate or alter application UI labels, database values, Google Sheet values, public PiniaStore/Function/Input names, table names, column names, or status values. Localization belongs to the app layer; durable AI guidance should remain concise English for consistent future use.
